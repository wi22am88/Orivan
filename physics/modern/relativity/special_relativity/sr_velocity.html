🚀 جمع السرعات النسبي

لماذا c+c ≠ 2c في عالم النسبية — ولماذا لا يمكن تجاوز سرعة الضوء أبداً

// المفهوم
السؤال: لو صاروخ يسير بـ 0.9c وأطلق مسباراً بـ 0.9c — ما سرعة المسبار؟
نيوتن يقول: 0.9c + 0.9c = 1.8c — تجاوز الضوء! ❌
آينشتاين يقول: قانون جمع السرعات النسبي يعطي نتيجة أقل من c دائماً ✅
// مقارنة القانونين

❌ نيوتن (يفشل عند سرعات عالية)

جمع السرعات الكلاسيكي — بسيط لكنه غير صحيح عند v قريبة من c

✅ آينشتاين (دائماً صحيح)

القانون النسبي — يضمن عدم تجاوز c مهما كانت القيم
// حاسبة جمع السرعات
🚀
